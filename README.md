# PHINMA UPang Campus Map Navigation

A web-based navigation map for PHINMA UPang Campus.

## Features

- Basic Map Functionalities
- Location Tracker
- Search Functionality
- Path Navigation
- Building Floor Map
- Event Viewer
- Feedback and Report

## Link

https://phinma-upang-map.herokuapp.com/
